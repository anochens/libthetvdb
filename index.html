libthetvdb by jeffWelling

Ruby interface for Thetvdb.com's API
  • Example Usage

Dependencies

  • ruby1.8
  • Xmlsimple

Install/Use

git clone git://github.com/jeffWelling/libthetvdb.git

Include it in your program.

Bug Tracking

Bug tracking for libthetvdb is done entirely with ticgit in the git repo itself. To submit a bug, checkout the repo, use ticgit to file your bug, and submit a pull request (via email if need be).
